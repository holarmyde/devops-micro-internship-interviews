# Week 09 — Docker (Containerization)

Docker is the packaging layer under everything — Kubernetes runs containers, CI pipelines build them, and every microservice ships as one. Interview questions here go beyond "what is a container" to test production habits: multi-stage builds, layer caching, image security, and what you do when a container is behaving unexpectedly in production. Knowing how to write a good Dockerfile is table stakes for any DevOps role.

**Appears in:** all technical rounds for roles that mention containers, Kubernetes, or CI/CD. Dockerfile and image optimization questions are common even in senior system design rounds.
