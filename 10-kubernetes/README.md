# Week 10 — Kubernetes (Deploy & Scale)

Kubernetes is the operating system of modern infrastructure. Interviews test whether you understand how Kubernetes actually works — not just how to run `kubectl apply`. Probes, resource requests, and secret management are the topics that separate engineers who have run production workloads from those who have only followed tutorials. If you have deployed a multi-service application to Kubernetes, you have answers that no documentation can give you.

**Appears in:** mid-to-senior technical rounds for any role that mentions Kubernetes, containers, or cloud-native. Questions escalate from "what is a deployment" to "how do you handle secret rotation" as seniority increases.
