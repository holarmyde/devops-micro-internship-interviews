# Week 04 — AWS Cloud

AWS questions appear at every stage of a DevOps or cloud engineering interview. IAM is the most tested topic because it is the root of every security decision on AWS — who can do what, and why. Understanding the difference between roles, users, and policies, and how security groups actually filter traffic, is the baseline. Without it, everything else (EKS, RDS, Secrets Manager) has no foundation.

**Appears in:** all technical rounds for roles that mention AWS. IAM questions are universal; service-specific questions (EKS, RDS, Route 53) appear in mid-to-senior rounds.
