# Week 08 — CI/CD

CI/CD questions test your delivery pipeline thinking — how code gets from a commit to a running service safely and repeatably. Interviewers want to understand if you know where build and deploy are separated, how environment gates work, and what happens when a deploy fails. Whether the tool is Azure DevOps, GitHub Actions, or Jenkins, the concepts are the same.

**Appears in:** all levels of DevOps and platform engineering interviews. Senior roles expect you to discuss pipeline design decisions — not just "we have a pipeline."
