# Week 02 — Git & GitHub

Every DevOps workflow touches Git. Interview questions here are not about memorising commands — they test whether you understand what Git is actually doing. The difference between `rebase` and `merge` is not syntax; it is history hygiene and what happens to a shared branch. Understanding `fetch` vs `pull` is about working safely in a team. These questions take 60 seconds in an interview and reveal years of experience.

**Appears in:** early technical rounds and coding interviews, at all levels.
