# Week 03 — DevOps Lifecycle

The "what is DevOps" question sounds soft, but it tests your mental model. Interviewers are not looking for a definition — they want to know if you think in terms of feedback loops, deployment frequency, and recovery time. Knowing the DORA metrics by name is the baseline; being able to say what yours looked like on a real project is what lands you the role.

**Appears in:** opening conversations, cultural fit rounds, and any discussion about team process or engineering maturity.
